数据可视化

1.散点图

以一个变量为横坐标，另一个变量为纵坐标，利用散点的分布形态反映变量关系的一种图形

plot(x, y, ‘.’, color=(r,g,b) )

plt.xlabel(‘x轴标签’)

plt.ylabel(‘y轴标签’)

plt.grid(True)

参数： X轴和Y轴的序列，大小点

2.折线图

趋势图，用直线段将各数据点连接起来组成的图形，以折线方式显示数据的变化趋势

plot(x, y, style, color, linewidth)

title(‘图的标题’)

style： - – -. : o , * ^ >

3.饼图

圆形图，划分为几个扇形的圆形统计图，能直观反映个体与总体的比例关系

pie(x, labels, colors, explode, autopct)

参数：绘图序列，各部分标签，各部分颜色，需要突出块状序列，占比显示格式 %.2f

4.柱状图

以长方形的单位长度，根据大小绘制的统计图，用来比较两个或以上的数据

bar(left, heigth, width, color)

barh(bottom, width, height, color) 横向

参数：

​ left x轴位置序列， arange函数产生

​ height y轴数值序列

￼￼

5.直方图

用一系列等宽不等高的长方形来绘制，宽度表示数据范围的间隔，高度表示在给定间隔内数据出现的频数，变化的高度形态表示数据的分布情况

hist(x, color, bins, cumulative=False)

参数：

​ x 需要进行绘制的向量

​ bins 设置直方图的分组个数

​ cumulative 设置是否累积计数，默认False

数据分析

基本统计分析

描述性统计分析，一般统计某个变量的最小值、第一四分位值、中值、第三四分位值、以及最大值

描述性统计分析函数

describe()

统计指标：计数、求和、平均差、方差、标准差

size、sum、mean、var、std

分组分析

根据分组字段、将分析对象划分为不同部分，以进行对比分析各组之间的差异性

groupby(by = [分组列1， 分组列2, … ])

[统计列，…].agg({统计列别名1：统计函数1，… })

分布分析

指根据分析目的，将数据（定量数据）进行等距或者不等距的分组，进行研究各组分布规律的一种分析方法

交叉分析

分析两个或以上分组变量之间的关系，以交叉表形式进行变量间关系的对比分析

pivot_table(values, index, columns, aggfunc, fill_value)

参数：数据透视表的值 ，行，列，统计函数，NA值的统一替换

返回值：数据透视表的结果

结构分析

在分组的基础上，计算各组成部分占比的比重，进行分析总体的内部特征的一种分析方法

axis参数： 0 - 按列运算，按行运算

数据框外运算：add、sub、multiply、div

数据框内运算：sum、mean、var、std

相关分析

研究对象之间是否存在某种依存关系，探讨相关方向以及相关程度

相关系数： 0 0.3 0.8 1 低中高

DataFrame.corr()

Series.corr(other)
